# 📊 Data Engineering Portfolio

Repositorio de aprendizaje progresivo en ingeniería de datos,
siguiendo un roadmap estructurado desde SQL avanzado hasta
arquitecturas modernas.

## 🛠️ Stack tecnológico

- **Base de datos:** PostgreSQL 16
- **Entorno:** Docker + pgAdmin
- **Próximamente:** dbt, Apache Spark, Kafka

## 🗺️ Roadmap

| Nivel | Tema                         | Estado         |
| ----- | ---------------------------- | -------------- |
| 1     | SQL Avanzado                 | ✅ Completo    |
| 2     | Internals de BD Relacionales | 🔄 En progreso |
| 3     | Modelado de Datos            | ⏳ Pendiente   |
| 4     | Cloud Warehouses + dbt       | ⏳ Pendiente   |
| 5     | NoSQL                        | ⏳ Pendiente   |
| 6     | Arquitecturas Modernas       | ⏳ Pendiente   |

## 🚀 Proyectos destacados

### [Dashboard de Análisis de Ventas](./nivel-1-sql-avanzado/proyecto-1-analisis-ventas)

Análisis de comportamiento de clientes y rendimiento geográfico
usando window functions, ROLLUP y LATERAL JOIN.

**Conceptos:** Window functions · ROLLUP · PIVOT · LATERAL JOIN

---

### [Motor de Recomendación SQL](./nivel-1-sql-avanzado/proyecto-2-motor-recomendacion)

Sistema de recomendación de películas construido íntegramente
en SQL con análisis de cohortes y reportes pivotados.

**Conceptos:** LATERAL JOIN · NOT EXISTS · Análisis de cohortes · PIVOT dinámico
